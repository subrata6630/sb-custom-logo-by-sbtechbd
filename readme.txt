A simple plugin to customize the WordPress login logo with an option to upload it via the admin panel.

== Description ==

Sb custom Logo by sbtechbd allows you to change the default WordPress login logo to your custom logo, with an easy-to-use interface that lets you upload the logo directly from the admin panel.

=== Features ===
* Change the WordPress login logo with your custom logo.
* Upload the logo through the WordPress media uploader.
* Simple and lightweight, no coding required.
* Fully compatible with the latest WordPress versions.

== Installation ==

1. Upload the plugin files to the `/sb-content/plugins/sb-custom-logo-by-sbtechbd` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to "Settings" > "sb Custom Logo" in the WordPress admin to configure the plugin.
4. Upload your custom logo and save the settings.

== Frequently Asked Questions ==

= How do I revert to the default WordPress logo? =
Simply deactivate the plugin, and the default WordPress logo will be restored.

= What image formats are supported? =
The plugin supports all standard image formats such as JPEG, PNG, and GIF.

= Is there a recommended logo size? =
It's recommended to use a logo with dimensions similar to the default WordPress logo (approximately 80x80 pixels) for optimal display.
